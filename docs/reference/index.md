## Conventions

Throughout the reference documentation:

- `<required>` - Required parameter
- `[optional]` - Optional parameter
- `|` - Alternative options
- `...` - Repeated elements

**Example**:
```bash
kassie tui [--profile <name>] [--server <address>]
```

This means both `--profile` and `--server` are optional parameters.
